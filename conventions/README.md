# Conventions for AI Agents and Contributors

This folder holds **English** convention documents for Cursor and other AI coding agents.

The canonical operational rules remain in the repository root:

| Document | Language | Purpose |
|----------|----------|---------|
| [`AGENTS.md`](../AGENTS.md) | Spanish | Short, imperative rules for agents working in this repo |
| [`conventions/AGENTS.en.md`](AGENTS.en.md) | English | Same rules, English translation |
| [`backend/docs/ai_workflow.md`](../backend/docs/ai_workflow.md) | Spanish | Methodology and session continuity |
| [`conventions/ai_workflow.en.md`](ai_workflow.en.md) | English | Same methodology, English translation |

When adding or changing operational rules, update **both** the Spanish source and the English mirror in this folder.

Technical contracts (parser, batch runner, schema) stay in `backend/docs/` and are not duplicated here.
